# vue-dialog-system 智慧医疗系统

## Project setup 安装依赖 
```
yarn install
```

### Compiles and hot-reloads for development 启动项目命令
```
yarn serve
```

### Compiles and minifies for production 打包项目命令
```
yarn build
```

### Lints and fixes files eslint  检查
```
yarn lint
```

### 关于项目目录介绍 
> node_modules 项目依赖包
> public 项目入口index.htm
> src 项目主要代码
    >> api 接口请求请求
    >> assets 静态资源
    >> common 公共js文件
    >> components 项目全局组件
    >> data 测试json文件
    >> router 路由目录
    >> store 全局状态管理
    >> styles 全局样式
    >> views 项目页面
    >> App.vue 主入口view
    >> main.js 主入口js
> vue.config.js vue 项目配置 例: **端口号,打包路径,webpack配置**
> packege.json 包管理/运行命令
